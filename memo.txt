仕様
１桁と１桁の式と答えが出され、それに合う演算子を答える四択クイズゲーム
1  1 = 2  // +
3  3 = 9 // * みたいな

左辺のオペランドは必ず１桁とする

左下に+-x÷ 左上に問題　右全部にスコアや時間　ボーナス情報などを表示

説明文字　→　よーいドン　→　スタート　→　ぬわあああんツカレタモン　で終了


出題の仕様
引数なし
事前に有効な式リストを作成する（答えがマイナス、割り切れない数にならない　答えが整数のみ）

{
  '+': [
    { 'left': 3, 'operator': '+', 'right': 3, 'calculated': 6 },
    ,,,
  ],
  '-': [
    ...
  ]
}

みたいにして返す

解答システム
個別の演算子をクリッカブルにしてハンドラを設定できるようにする 〆
↓
クリックされた演算子を取得 〆
↓
problemのcurrentの演算子とクリックされた演算子を比較 〆
↓
等しいならスコアアップして音を慣らす　間違いなら不正解サウンドだけを慣らす 〆
↓
問題エンティティを削除して次の問題をshowする 〆
